# Task 3: Data_Handling_with_pandas & Titanic Analysis

This project demonstrates core data manipulation skills using **Python** and **Pandas**. It is divided into two primary sections:

## 1. Custom Dataset: ASTU Student Records
- **Objective**: Build a structured dataset from scratch using a Python dictionary.
- **Data**: 15 student records from **Adama Science and Technology University (ASTU)**.
- **Features**: Includes names (Milkesa, Hawwi, etc.), engineering departments, specific UGR ID formats, and CGPA.
- **Analysis**: Calculated average performance (CGPA) per engineering department.

## 2. Real-World Analysis: Titanic Survivors
- **Data Cleaning**: Handled missing values (Age median, Embarked mode) and removed the unnecessary `Cabin` feature.
- **Key Findings**:
  - **Gender**: Females had a significantly higher survival rate (~74%).
  - **Class**: 1st Class passengers were prioritized, showing the impact of socio-economic status on survival.
  - **Demographics**: Children were prioritized over adults during the evacuation.
- **Filtering**: Identified high-probability survival groups (1st Class Females).

## Technologies Used
- **Python 3.x**
- **Pandas**: For data cleaning and grouping.
